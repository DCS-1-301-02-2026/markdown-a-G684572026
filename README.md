#　拓殖大学　情報工学科の紹介:rocket:
##　学科の概要
情報工学科では、高度な**プログラミング技術**や*ネットワーク技術*を基礎から学びます。
### 主な学習分野
- 人工知能
- システムエンジニアリング
- セキュリティ

### カリキュラムの流れ
1. 1年次：プログラミングの基礎
2. 2年次：データ構造とアルゴリズム
3. 3,4年次：卒業研究
